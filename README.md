## Abstract

Innovest fills the communication gap between emerging entrepreneurs and investors by creating a platform for collaboration, feedback, and investment. Our platform offers a space for startups to showcase their products, receive feedback, and discover potential investors. Investors can find promising startups and provide mentorship, while an AI-powered chatbot assists users in navigating the platform.

## Features

- **Video Conferencing:** Facilitates seamless video communication for meetings and collaborations.
- **AI Chatbot:** Provides virtual assistance and personalized advice to users.
- **Live Streaming:** Broadcasts live events and product launches.
- **Posts:** Allows users to share updates, news, and content.
- **Product Testing:** Enables public users to test and provide feedback on startup products.
- **Interactive Dashboard:** Provides real-time insights and tracks user activities.
- **Collaboration Tools:** Enhances teamwork and project management.

## Introduction

Innovest serves as a platform for connecting startups, investors, governmental organizations, and general users. Entrepreneurs can pitch their ideas to investors, startups can collaborate with public users, and everyone has access to an interactive dashboard for tracking activities. The platform also includes a conference portal for online meetings and events, as well as a social media-like functionality for sharing thoughts and updates.

## Problem Statement

There is a significant gap between creative ideas and their development into successful businesses. Startups struggle with obtaining finance, gathering diverse feedback, and forming collaborations. Existing platforms often focus on specific functions, leaving startups without a comprehensive ecosystem that combines financial support, public engagement, and professional mentoring. Innovest aims to create an integrated digital platform that brings together entrepreneurs, investors, and the general public to foster an entrepreneurial ecosystem.

## Getting Started

### Prerequisites

- Node.js
- npm or yarn
- MongoDB

### Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/innovest.git
    ```
2. Navigate to the project directory:
    ```bash
    cd innovest
    ```
3. Install dependencies:
    ```bash
    npm install
    ```
    or
    ```bash
    yarn install
    ```

### Running the Application

1. Start the development server:
    ```bash
    npm start
    ```
    or
    ```bash
    yarn start
    ```
2. Open your browser and navigate to `http://localhost:3000`
